📖 操作方法 / How to Play

🔥 フェーズ① 鍛造 / Forging

  • 左グリップ 鉄を掴む／離す / Hold iron
  • 炉に入れると加熱 / Heat iron in furnace
  • 80〜60° 〇 適正 / Optimal — Perfect/Good/Miss
  • 100〜80° △ 熱すぎ / Too hot — Good/Miss only
  • 60°未満 × 冷めすぎ / Too cold — Miss only
  • 作業台に置いて 右トリガー で打撃 / Hammer on bench
  • 30回打撃でフェーズ終了 / 30 hits to complete

🔨 フェーズ② 成形 / Shaping

  • 右トリガー 赤いゾーンを叩く / Hammer red zones
  • 左Xボタン 刃物を表裏反転 / Flip blade
  • 表(+)歪みは表から、裏(-)歪みは裏から / Hit matching side
  • 歪み0→100点、未修正→0点 / 0 distortion = 100pts

⚔️ フェーズ③ 研ぎ / Sharpening

  • 両手グリップ 刃物を保持 / Hold with both grips
  • 前後にスライド / Slide blade back & forth
  • 目標角度 -15°（コントローラーを少し手前に傾ける）
  • Target -15° — tilt controller slightly toward you
  • 両手の角度を揃えること / Keep both hands aligned
  • 「シャーッ」音が正解 / Shhh sound = correct angle
  • 30ストロークで完了 / 30 strokes to complete

🎮 共通 / Common

  • 右Aボタン 次のフェーズへ / Next phase
  • 右Bボタン 現在フェーズをリトライ / Retry phase
