рҹҸҶ гӮ№гӮігӮўиЁҳйҢІ
рҹ“– ж“ҚдҪңж–№жі• / How to Play

рҹ”Ҙ гғ•гӮ§гғјгӮәв‘  йҚӣйҖ  / Forging

  • е·ҰгӮ°гғӘгғғгғ— йү„гӮ’жҺҙгӮҖпјҸйӣўгҒҷ / Hold iron
  • зӮүгҒ«е…ҘгӮҢгӮӢгҒЁеҠ зҶұ / Heat iron in furnace
  • 80гҖң60В° гҖҮ йҒ©жӯЈ / Optimal вҖ” Perfect/Good/Miss
  • 100гҖң80В° в–і зҶұгҒҷгҒҺ / Too hot вҖ” Good/Miss only
  • 60В°жңӘжәҖ Г— еҶ·гӮҒгҒҷгҒҺ / Too cold вҖ” Miss only
  • дҪңжҘӯеҸ°гҒ«зҪ®гҒ„гҒҰ еҸігғҲгғӘгӮ¬гғј гҒ§жү“ж’ғ / Hammer on bench
  • 30еӣһжү“ж’ғгҒ§гғ•гӮ§гғјгӮәзөӮдәҶ / 30 hits to complete

рҹ”Ё гғ•гӮ§гғјгӮәв‘Ў жҲҗеҪў / Shaping

  • еҸігғҲгғӘгӮ¬гғј иөӨгҒ„гӮҫгғјгғігӮ’еҸ©гҒҸ / Hammer red zones
  • е·ҰXгғңгӮҝгғі еҲғзү©гӮ’иЎЁиЈҸеҸҚи»ў / Flip blade
  • иЎЁ(+)жӯӘгҒҝгҒҜиЎЁгҒӢгӮүгҖҒиЈҸ(-)жӯӘгҒҝгҒҜиЈҸгҒӢгӮү / Hit matching side
  • жӯӘгҒҝ0вҶ’100зӮ№гҖҒжңӘдҝ®жӯЈвҶ’0зӮ№ / 0 distortion = 100pts

вҡ”пёҸ гғ•гӮ§гғјгӮәв‘ў з ”гҒҺ / Sharpening

  • дёЎжүӢгӮ°гғӘгғғгғ— еҲғзү©гӮ’дҝқжҢҒ / Hold with both grips
  • еүҚеҫҢгҒ«гӮ№гғ©гӮӨгғү / Slide blade back & forth
  • зӣ®жЁҷи§’еәҰ -15В°пјҲгӮігғігғҲгғӯгғјгғ©гғјгӮ’е°‘гҒ—жүӢеүҚгҒ«еӮҫгҒ‘гӮӢпјү
  • Target -15В° вҖ” tilt controller slightly toward you
  • дёЎжүӢгҒ®и§’еәҰгӮ’жҸғгҒҲгӮӢгҒ“гҒЁ / Keep both hands aligned
  • гҖҢгӮ·гғЈгғјгғғгҖҚйҹігҒҢжӯЈи§Ј / Shhh sound = correct angle
  • 30гӮ№гғҲгғӯгғјгӮҜгҒ§е®ҢдәҶ / 30 strokes to complete

рҹҺ® е…ұйҖҡ / Common

  • еҸіAгғңгӮҝгғі ж¬ЎгҒ®гғ•гӮ§гғјгӮәгҒё / Next phase
  • еҸіBгғңгӮҝгғі зҸҫеңЁгғ•гӮ§гғјгӮәгӮ’гғӘгғҲгғ©гӮӨ / Retry phase
